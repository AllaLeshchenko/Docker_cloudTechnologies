1. Что такое Docker? - 
 b) Контейнерная платформа для упаковки, доставки и запуска приложений.

2.Соотнесите описание(поданы ниже) и понятие (Docker, Классическая виртуализация):
- Каждая виртуальная машина имеет свою собственную операционную систему => Классическая виртуализация;
- Все контейнеры на одном хосте используют общую операционную систему => Docker;
- Запускает полные копии операционной системы => Docker;
- Запускает полные копии операционной системы => Классическая виртуализация;
- Использует контейнеризацию для упаковки и запуска приложения => Docker;
- Использует гипервизор для виртуализации аппаратных ресурсов => Классическая виртуализация.

3.  Верно ли утверждение:
Все контейнеры, запущенные на одной машине, делят ядро операционной системы, поэтому они стартуют немедленно и более эффективно.
а) Да

4. К какому из компонентов Docker относится описание:
Изолированные среды, которые запускаются на основе образов и могут содержать всё необходимое для выполнения приложения.
c) Docker Container

5. Что представляет собой Dockerfile?
a) Текстовый файл, описывающий шаги по созданию Docker образа.

6.  Что представляют собой Docker Images?
b) Шаблоны для создания контейнеров, состоящие из слоев

7. Что такое Docker Hub и другие Docker реестры (registries)?
b) Онлайн-платформа для обмена, хранения и управления Docker образами.
